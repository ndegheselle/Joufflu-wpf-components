Clone the library :
```
git clone https://github.com/tree-sitter/csharp-tree-sitter.git --recursive
```

Open a `Developer PowerShell for VS` in the `./csharp-tree-sitter/tree-sitter` folder and build the dependencies :
```
nmake
```

Build the C# project `./csharp-tree-sitter/csharp-tree-sitter.csproj`
